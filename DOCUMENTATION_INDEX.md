## üîë Key Concepts

### 7-Step Workflow
1. **Batch Creation** - Validate & store metadata
2. **AES Encryption** - Encrypt batch details (AES-256)
3. **DataHash** - Generate fingerprint (SHA-256)
4. **Hash-Chain** - Create chain link (SHA-256 concatenation)
5. **QR Code** - Generate physical code (batchId|chainHash)
6. **HMAC Signature** - Sign event (HMAC-SHA256)
7. **Response Object** - Return security package

### Security Layers
- **Confidentiality**: AES-256 encryption
- **Integrity**: SHA-256 hashing
- **Tamper Evidence**: Hash-chain linking
- **Authentication**: HMAC signatures
- **Physical Tracking**: QR codes

### API Endpoints
- `POST /api/manufacturer/create-batch` - Create batch
- `GET /api/manufacturer/batch/:batchId` - Get batch
- `POST /api/manufacturer/verify-batch` - Verify integrity
